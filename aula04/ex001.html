JavaScript primeiros passos.

Variáveis

Variáveis são como uma caixa, que serve para guardar algo, elas guardão valores, outro exemplo, é que elas podem ser como uma vaga em um estacionamento, que irá receber um carro, logo ele estará guardado ali.

Elas podem ser declaradas de 3 formas em JavaScript : var, let e const.

Obs: const não pode ter seu valor alterado depois de declarado, ao contrário de var e let.

Escopo

O escopo em javascript define a limitação e visibilidade de um bloco de código.

Escopo Global: quando a variável é declarada fora de qualquer bloco, sua visibilidade fica disponível em todo o código.

Escopo local: quando a variável é declarada dentro de um bloco, sua visibilidade pode ficar disponível ou não.

Arrays

São um tipo de lista, ou matriz de variáveis.

Um array, é como uma caixa com várias outras caixas dentro.

Sintáxe de declaração array exemplificado : let array = ['string', 1, true].

Como você pode observar, esses valores podem ser de vários tipos.

Array deve ser declarado entre "[]" colchetes e podem guardar dentro inclusive outros arrays como valores.

Ex: let array = ['string', 1, true, false, ['array1'], ['array2']].

Manipulando Arrays

Ao ser declarado, um array tras consigo uma série de formar de manipulá-lo.

  • forEach() - intera um array;
  • push() - add item no final do array;
  • pop() - remove item no final do array;
  • shift() - remove item no início do array;
  • unshift() - add item no início do array;
  • indexOf() - retorna o índice de um valor;
  • splice() - remove ou substitui um item pelo índice;
  • slice() - retorna uma parte de um array existente;

Objetos

São dados que possuem propriedades e valores que definem suas características. Deve ser declarado entre chaves "{}".

Imagine uma xícara, ela tem cor, pode ter vários tamanhos e funções.

pode ser declarada assim:

var xicara = {

cor: 'azul',

tamanho: 'p',

funcao: tomarCafe()

}

Manipulando Objetos

As propriedades de objetos podem ser atribuídas à variáveis, facilitando a manipulação do objeto. chamamos isso de desestruturação.

var cor = xicara.cor;

var tamanho = xicara.tamanho;

var funcao = tomarCafe();
